# search-cpf
Procura de CPF na base da Receita

## Uso
faça uma requisição na rota http://localhost:3000/api/v1/cpf/"cpf"/"data_nascimento"

***Cpf: 00000000026 <-- dessa forma  <br/>
data_nascimento: 18081986 <-- dessa forma <br/>***
